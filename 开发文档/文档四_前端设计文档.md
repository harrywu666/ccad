# 文档四：前端设计文档
## 室内装饰施工图 AI 自动审核系统

**版本：V1.1 | 日期：2026年2月**
**更新：全面使用shadcn/ui组件库，新增项目管理/分类/数据替换相关界面**

---

## 一、技术规范

### 1.1 组件库
全站使用 **shadcn/ui**，主要用到的组件：
- `Button` — 所有按钮
- `Input` — 输入框
- `Select` — 下拉选择
- `Table` — 数据表格
- `Dialog` — 弹窗
- `AlertDialog` — 二次确认弹窗
- `Alert` — 页面提示
- `Badge` — 状态标签
- `Card` — 卡片容器
- `Tabs` — 标签页切换
- `Progress` — 进度条
- `Tooltip` — 悬浮提示
- `ScrollArea` — 可滚动区域

### 1.2 端口
- 前端：http://localhost:7001
- 后端API：http://localhost:7000

### 1.3 颜色规范
| 用途 | 颜色 | shadcn变量 |
|------|------|-----------|
| 主色 | 蓝色 | `bg-primary` |
| 错误/严重 | 红色 | `text-destructive` |
| 警告 | 黄色 | `text-yellow-500` |
| 成功/通过 | 绿色 | `text-green-600` |
| 待处理 | 灰色 | `text-muted-foreground` |

## 十、全局交互规范

### 加载状态
- 所有异步操作：shadcn Button显示loading spinner
- 禁止重复点击

### 错误处理
- API失败：页面顶部 shadcn Alert（红色）
- 上传失败：上传区显示错误信息 + 重试按钮

### 空状态
- 项目列表为空：居中显示插图 + 「创建第一个项目」按钮
- 某分类无项目：显示「此分类暂无项目」
- 无审核问题：对应标签页显示 shadcn Alert（绿色）「✅ 未发现此类问题」

### 数据版本标识
有新版本数据时，用 shadcn Badge 标注：
- `v2（最新）` — 蓝色
- `v1（历史）` — 灰色

### 响应式
- 主要支持桌面端（1280px+）
- 平板（768px+）基本可用
