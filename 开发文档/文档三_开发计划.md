### 任务5-2：视口坐标换算

打开Trae，输入：
```
帮我在services/dxf_service.py中实现视口坐标换算函数：

函数 calc_model_range(viewport) -> dict
  从ezdxf的VIEWPORT实体中计算该视口对应的模型空间范围
  
  viewport实体的关键属性：
    viewport.dxf.view_center_point  → 模型空间中心点 (x, y)
    viewport.dxf.view_height        → 视口在模型空间中的高度
    viewport.dxf.width              → 视口在布局空间中的宽度（paper units）
    viewport.dxf.height             → 视口在布局空间中的高度（paper units）
    viewport.dxf.scale              → 视口比例（如0.02表示1:50）
    
  计算逻辑：
    model_height = viewport.dxf.view_height
    model_width  = model_height * (viewport.dxf.width / viewport.dxf.height)
    cx, cy = viewport.dxf.view_center_point
    
    return {
      "min": [cx - model_width/2,  cy - model_height/2],
      "max": [cx + model_width/2,  cy + model_height/2]
    }

函数 get_visible_layers(viewport) -> set[str]
  获取该视口中可见的图层名称集合
  视口有独立的图层冻结状态，需要从viewport的frozen_layers属性读取
  同时过滤掉doc.layers中is_on()==False或is_frozen()==True的全局冻结图层
  返回：所有在该视口中可见的图层名称集合
```

### 任务5-3：单布局数据提取

打开Trae，输入：
```
帮我在services/dxf_service.py中实现单布局数据提取函数：

函数 extract_layout(doc, layout_name: str, dwg_filename: str) -> dict
  用ezdxf提取指定布局的所有数据，返回符合文档二第六节格式的JSON字典

  参数：
    doc：ezdxf.readfile()返回的文档对象
    layout_name：布局名称（如"平面布置图"）
    dwg_filename：来源DWG文件名（用于记录）

  提取步骤：

  1. 获取布局对象
     layout = doc.layouts.get(layout_name)
     如果不存在，返回None

  2. 找到布局中的主视口（面积最大的那个VIEWPORT）
     all_vps = list(layout.query("VIEWPORT"))
     过滤掉 vp.dxf.id == 1 的（id=1是布局本身的虚拟视口，不是真实视口）
     选择 dxf.width * dxf.height 最大的视口作为主视口
     调用 calc_model_range(main_vp) 得到 model_range
     调用 get_visible_layers(main_vp) 得到 visible_layers

  3. 提取DIMENSION（尺寸标注）
     从模型空间（doc.modelspace()）查询DIMENSION：
       for dim in doc.modelspace().query("DIMENSION"):
         如果 dim.dxf.layer 不在 visible_layers 中，跳过
```
